现代 VPN 客户端 Android 版

一个提供加密连接的 Android App，是用 Flutter 开发的。App 使用本地 VPN 和远端服务器之间的加密通道来加密设备的网络流量，框架可以开放支持多种协议。

开发语言：Dart，Java

关键词：Flutter，Room Dao，VPN，SOCKS，Shadowsocks，V2Ray，CameraX，ZXing，MLKit

加密连接工具 Windows 端

Windows 加密代理工具 App，是 Windows .NET WPF 桌面应用程序。软件自动查找可用端口并提供系统流量代理服务，也支持通过 HTTP API 访问代理服务。App 没有使用数据库，而是用了 XML 序列化、反序列化的方式来存取数据。

开发语言：C#

关键词：WPF，XAML，MVVM，Proxy，SOCKS，Shadowsocks，V2Ray，ZXing，XML
