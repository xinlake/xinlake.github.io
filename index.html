Privch 0.9.0

XTransmit 项目更名为 Privch，因为 XTransmit 这个名称的 com 域名已经被注册了。我是 2019 年 08 月份开源的，10 月份发了第一个 Release，前不久准备注册域名查了一下发现 xtransmit.com 已经在 2019 年 12 月被人注册了。注册商显示为 godaddy，我通过 godaddy 尝试和域名注册者取得联系，问对方是否愿意转让，可是没有得到任何回应，无奈只好放弃原名称。不过，新的名称 Privch 意义更为准确，也更加友好。

RECENT WORKS

加密连接工具 Android 端

一个提供加密连接的 Android App，是用 Flutter 开发的，目前没有开源。App 使用本地 VPN 和远端服务器之间的加密通道来加密设备的网络流量，框架可以开放支持多种加密协议。

开发语言：Dart，Java

关键词：Flutter，Room Dao，VPN，SOCKS，Shadowsocks，V2Ray，CameraX，ZXing，MLKit

加密连接工具 Windows 端

Windows 加密代理工具 App，是 Windows .NET WPF 桌面应用程序。软件自动查找可用端口并提供系统流量代理服务，也支持通过 HTTP API 访问代理服务。App 没有使用数据库，而是用了 XML 序列化、反序列化的方式来存取数据。

开发语言：C#

关键词：WPF，Proxy，SOCKS，Shadowsocks，V2Ray，ZXing，XML
