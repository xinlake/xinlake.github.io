Privoxy Windows
  • 2019-10-19
  • Network

Privoxy 是具有高级过滤功能的非缓存 Web 代理（同类软件里分缓存和非缓存两种），可增强隐私性。Privoxy 常常作为一个辅助的流量转发模块，工作于操作系统代理接口和 SOCKS5 代理服务接口之间，将本地 HTTP 流量转发到本地或远端 SOCKS5 服务器，从而让 Web 应用可以直接使用 SOCKS5 代理且不需要任何设置，被广泛应用在各种连接国际网络的工具里。
